# autowatering
Code and wiring instructions for my first Automated Watering System Prototype
All instructions are included in the FIRSTWORKING.ino file.

Materials Needed:
1. Arduino Uno (I used Arduino Uno R4 WIFI)
2. Mini Submersible Water Pump DC 3-5v with Tubing
3. 1 Channel 5v Relay Module
4. Water Sensors for Arduino
5. I2C IIC LCD 1602 Module, 16x2 Display
6. Breadboard
7. Jumper cables (Male to Male, F-M, F-F)
8. Tape to attach sensors to tray
9. Water tray
10. Water resevoir
11. USB-C Cable
12. Free time

*NOTE- the LCD screen code is very buggy at the moment - it gets jumbled every time the system detects a change in state. Still needs fixing
*Ensure that the water tray end of the tubing is not submerged - this may cause a siphoning effect that repeatedly causes the water to return to the resevoir after filling
